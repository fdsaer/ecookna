## Файлы настроек

```
.
└─ config
  ├─ app.settings         # Подключение к couchdb, префикс, суффикс, зона
  ├─ env                  # Переменные окружения для webpack
  ├─ paths                # Пути к файлам и папкам для webpack
  ├─ polyfills            # Полифилы для старых браузеров
  ├─ webpack.config.dev   # Конфигурация для отладки
  ├─ webpack.config.prod  # Конфигурация для production
  └─ webpackDevServer     # Конфигурация отладочного http-сервера
```
