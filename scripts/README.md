## Скрипты для сборки и тестирования

```
.
└─ scripts
  ├─ start          # Компилирует dev-сборку и запускает локальный сервер для отладки
  ├─ build          # Компилирует prod-сборку для публикации на вебсервере
  ├─ analyze        # Запускает webpack-bundle-analyzer
  ├─ test           # Запускает автоматические тесты
  └─ meta.link      # Копирует dev-версию файлов в node_modules (для отладки библиотек)
```
